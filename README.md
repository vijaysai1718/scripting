# scripting

**steps:**

* Create repo in Github
* Clone repo in local laptop
* Start developing
* Add the files to temp area
```
git add <file_name>
```
* Commit the changes
```
git commit -m "addeed the new file"
```
* Push the changes 
```
git push origin main
```
